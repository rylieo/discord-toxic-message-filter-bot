# Discord Toxic Message Filter Bot

Bot ini dibuat untuk menghapus pesan tidak layak atau toxic di Discord secara otomatis.

## Fitur

- Menghapus pesan yang mengandung kata-kata toxic seperti, "ya*im", "pi*tu", dan lainnya anda bisa menambahkan kata lain di scriptnya.

## Note

- Python 3.8 atau lebih tinggi.

## Instalasi

- cmd: pip install -r requirements.txt
- tambahkan bot ke server/komunitas discord anda
- run: python bot.py
